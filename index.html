Help & Security Notes

SecurePass Lite is offline-first. No server is used.

Storage

Your data is stored in your current browser’s localStorage. Clearing site data or using private mode may remove it.

Clipboard

Copied passwords may be readable by other apps on your device. Clear your clipboard if needed.

QR Code Safety

A QR code is equivalent to showing the password in plaintext. Only use it in private.

Settings

Export or import your local password book.
Data Backup
Export downloads a file. Import overwrites your current data.
